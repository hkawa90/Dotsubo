### Ethernet II

``` title="Ethernet II frame"
|  8bytes  |    6bytes   |    6bytes   |2bytes|46-1500bytes|4bytes|
+----------+------+-------------+-------------+------+------------+------+
| Preamble | Dst Address | Src Address |  Type|    Data    |  FCS |
           |<--------------------------------->
                        Ethernet header
```
